# Product

## Register

product

## Users

Administradores de TI que gerenciam a infraestrutura da organização — computadores (Active Directory + WMI), impressoras (SNMP) e usuários do AD. Usam o sistema diariamente para monitorar status, sincronizar dados e manter a rede operacional. Contexto de trabalho: agilidade e confiabilidade são críticas; cada segundo conta durante uma investigação.

## Product Purpose

Sistema web para centralizar o gerenciamento de ativos de TI — computadores sincronizados via Active Directory, impressoras monitoradas via SNMP, e usuários do AD gerenciados diretamente. Elimina a necessidade de ferramentas separadas para cada tarefa, oferecendo um painel unificado e em tempo real do estado da infraestrutura.

## Brand Personality

Preciso, técnico, confiável. Aparência moderna e técnica, sem firulas — uma ferramenta de trabalho que transmite seriedade e competência.

## Anti-references

Nenhuma referência específica. Deve evitar visual "brinquedo" ou excessivamente decorativo — a interface é uma ferramenta profissional.

## Design Principles

1. **Ferramenta, não vitrine** — A interface prioriza funcionalidade e agilidade sobre decoração. Cada elemento existe para fazer o trabalho do admin mais rápido.
2. **Clareza técnica** — Dados técnicos (IPs, status, modelos) são apresentados de forma direta, sem ruído visual. O admin precisa entender o estado da infraestrutura em um relance.
3. **Consistência sobre surpresa** — Padrões visuais e de interação são uniformes em todas as páginas. O admin nunca precisa reaprender ao navegar.
4. **Feedback imediato** — Toda ação (sincronizar, testar, salvar) produz feedback claro de sucesso ou erro, sem ambiguidade.

## Accessibility & Inclusion

WCAG AA como padrão: contraste mínimo 4.5:1 para texto normal, foco visível em todos os elementos interativos, suporte a navegação por teclado.
